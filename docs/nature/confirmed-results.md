# 已确认单只精灵性格结论台账

本文件记录用户确认过的单只精灵性格结论，作为规则回归基线。只有用户明确确认的最终分档可以写入；草稿或仍有争议的结论应留在会话记录或 `docs/nature/open-issues.md`。

## 使用规则

1. 每次修改通用性格规则后，重新检查本文件中的全部确认结果。
2. 分档与本基线不一致时暂停发布，先判断是预期规则变化、资料变化还是回归问题。
3. 只有用户重新确认后才能更新已有分档。
4. 本文件是人工维护台账，不由脚本生成。
